# Build Your Bot a Brain — Shot List v1 (Short)

## Format
- Aspect: 9:16 primary (safe-center for cross-post)
- Duration: 60–90s
- Style: fast cuts, terminal/UI overlays, kinetic captions

## Shots by segment

### 1) Hook (0:00–0:08)
- Tight shot: chaotic chat threads / repeated prompts.
- Smash cut to bold title card: "BUILD YOUR BOT A BRAIN".
- Quick UI flash of Carapace control surface.

### 2) Problem (0:08–0:20)
- On-screen text: "Chat-only bots forget."
- Visual: same question being retyped in different sessions.
- Subtle red warning overlays: "No persistence" / "No compounding".

### 3) Shift framework (0:20–0:40)
- Four animated blocks appear sequentially:
  1. Memory
  2. Objectives
  3. Tools
  4. Routines
- Each block gets 1 quick supporting visual:
  - Memory: notes/timeline node stack
  - Objectives: checklist / target marker
  - Tools: API/wrench icon + action pulse
  - Routines: recurring cycle animation

### 4) Demo beat (0:40–1:05)
- Screen capture montage (2–3 second beats):
  - objective creation
  - session execution
  - window/tool action
  - result summary
- Overlay copy: "System > Prompt"

### 5) Close (1:05–1:20)
- Hero UI shot, clean and stable.
- Text sequence:
  - "You don’t need a better prompt"
  - "You need a better architecture"
  - "Build your bot a brain"
- End card: handle + "Full build walkthrough next"
